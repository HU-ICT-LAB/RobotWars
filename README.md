# RobotWars

## Installation
To install the package run the following command:
```bash
pip install .
```
If you are a developer you need to run this command:
```bash
pip install .[dev]
```

Start the api: 
```bash
cd RobotWarsApi && python3 api.py
```

Start the react app
```bash
cd robotwarsreact && yarn install && yarn start
```
